# 批量注释与反注释(转)
## 块选择模式
 **注释** 

* ctrl+v(Win下面ctrl+q)进入列模式
* 大写I，输入注释符“#”或“//”,按ESC重复步骤

**反注释**

* ctrl+v(Win下面ctrl+q)进入列模式，选中按d

## 替换命令
* 批量注释
:起始行号,结束行号s/^/注释符/g

* 取消注释：
:起始行号,结束行号s/^注释符//g

## 实例
> 在27 - 30行添加 // 注释
  :27,30s#^#//#g

> 在27 - 30行删除 // 注释
  :27,30s#^//##g

> 在10 - 20行添加 # 注释
  :10,20s/^/#/g
  
> 在10 - 20行删除 # 注释
  :10,20s/^/#/g

**注意例子中正则的分割符使用的是相反的符号，如果匹配// 那么使用 #作分隔符这样不需要对/作转义处理，节省输入次数。**
